#Car Repair Management System

This project is a Python-based Car Repair Management System developed as part of Programming Fundamentals. It is a menu-driven console application designed to manage basic operations of a car service center.

#Features
- Perform car service operations
- Generate customer receipts with cost and discounts
- Manage customer records
- View and update services and parts
- Simple and user-friendly menu interface

#Technologies Used
- Python (Core Programming)
- File handling / Data structures

#Project Structure
- Main program file
- Service and parts data handling
- Customer management logic

#How to Run
1. Install Python
2. Open project in VS Code
3. Run the main Python file
